# Security Notes

This application stores journal entries locally in encrypted form.

## Important

* Your encryption key is NOT recoverable.
* If you forget your key, journal contents cannot be decrypted.
* Keep backups secure.
* The project does not transmit journal data over the internet.

## Antivirus Notice

Because the executable is unsigned and performs:

* encryption
* filesystem operations
* console manipulation

some antivirus software may initially show warnings.

The source code is fully available for inspection.
